## 개발 환경

### 윈도우
1. 개발 환경
    * mingw를 통한 gcc 설치
1. 빌드
```
mkdir build
cd build
cmake ..
make
```

### 리눅스
1. 개발 환경
    * gcc
1. 빌드
```
sh ./scripts/build.sh
```
